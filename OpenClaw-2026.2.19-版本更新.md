## 二、iOS 体验三项优化

**远程唤醒**

通过 APNs 远程唤醒后台 iOS 节点。以前调用设备功能时节点没响应导致失败，这次从根源上解决。

**屏幕录制稳定性**

修复了 WKWebView 生命周期导致的崩溃 bug，录制中途不再莫名退出。

**配对流程**

解决了配对状态闪烁和重复配对的问题，首次连接体验更流畅。

## 三、新命令 + 安全加固

**设备管理新增三条命令：**

```bash
device.pair.remove       # 移除指定配对设备
openclaw devices remove  # 清理已配对设备列表
openclaw devices clear   # 清理待处理配对请求
```

**安全层同步加强：**

Gateway 认证默认启用 token 模式，自动生成并保存 token；新增未授权 HTTP 访问检测，浏览器中继认证机制同步强化。
